# Quickstart

    ```bash
    git clone
    cd SeleniumPlusPlus
    python -m venv .venv
    .venv\Scripts\activate
    pip install -r requirements.txt
    ```

## Poetry

> pip install poetry
> poetry init
> poetry config pypi-token.pypi TOKEN_VAR
> poetry build
> poetry publish --build


## Requirements

pip freeze > requirements.txt
